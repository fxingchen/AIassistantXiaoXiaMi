# 小虾米 v0.1.1 - 深度思考与生成状态

## 🎯 核心改进

小虾米现在拥有类似 CodeBuddy 的**深度思考**和**生成状态**提示，让用户清楚知道 AI 在做什么！

## ✨ 新增功能

### 1. 深度思考指示器

**位置**：消息气泡顶部
**状态**：🧠 深度思考中...

**效果**：
```
┌─────────────────────────────────────┐
│ 🧠 深度思考中...          [展开]   │
│  ─────────────────────────────────  │
│  ✓ 理解用户需求：帮我写个函数...  │
│  ✓ 分析问题类型和关键要素         │
│  ✓ 检索相关知识和上下文           │
│  → 正在组织答案结构...            │
└─────────────────────────────────────┘
```

**特点**：
- 可展开/收起，点击标题栏即可
- 显示思考步骤
- 紫色高亮，视觉区分

### 2. 生成状态提示

**位置**：思考块下方
**状态**：正在生成答案，请稍等...

**效果**：
```
┌─────────────────────────────────────┐
│ 🔄 正在生成答案，请稍等...          │
└─────────────────────────────────────┘
```

**特点**：
- 带旋转动画的加载图标
- 绿色主题，表示进行中
- 自动消失，完成后隐藏

### 3. 智能状态流转

**流程**：
```
1. 发送问题
   ↓
2. 🧠 深度思考中...（显示思考块）
   - 理解用户需求
   - 分析问题类型
   - 检索相关知识
   - 组织答案结构
   ↓
3. 🔄 正在生成答案...（显示生成状态）
   - 思考块隐藏
   - 生成状态显示
   - 实时流式输出答案
   ↓
4. ✅ 完成（状态消失）
   - 思考块隐藏
   - 生成状态隐藏
   - 显示最终答案
```

## 🔧 技术实现

### 1. 思考块 HTML 结构
```html
<div class="thinking-block">
  <div class="thinking-header" onclick="toggleThinking(this)">
    <div class="thinking-icon">🧠</div>
    <span class="thinking-title">深度思考中...</span>
    <span class="thinking-toggle">展开</span>
  </div>
  <div class="thinking-content">
    <!-- 思考步骤 -->
  </div>
</div>
```

### 2. 生成状态 HTML 结构
```html
<div class="generating-status">
  <div class="generating-spinner"></div>
  <span>正在生成答案，请稍等...</span>
</div>
```

### 3. 状态切换逻辑
```javascript
// 开始时
thinkingBlock.style.display = 'block';
generatingStatus.style.display = 'none';

// 收到第一个token时
thinkingBlock.style.display = 'none';
generatingStatus.style.display = 'flex';

// 完成时
thinkingBlock.style.display = 'none';
generatingStatus.style.display = 'none';
```

### 4. 展开收起功能
```javascript
function toggleThinking(header) {
  const block = header.closest('.thinking-block');
  block.classList.toggle('expanded');
}
```
