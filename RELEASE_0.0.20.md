# 小虾米 AI 助手 v0.0.20

发布目标版本：0.0.20

本次版本聚焦“最新信息检索”场景下的真实执行能力，重点解决两类问题：一类是模型口头说会搜索，却没有真正调用工具；另一类是把“今天 / 最新 / 近期 / 现在”错误改写成 2023、2024 之类年份，导致结果失真。

## 亮点

- 新增 `web_search` 工具链路，允许模型直接联网搜索后继续分析
- 如果模型只输出“我会先通过 web_search 搜索”之类说明文字，前端会自动追打一轮，强制它输出真实工具代码块
- 提示词现在明确要求保留用户原始时间词，不再擅自改写成具体年份
- 自测补充了“承诺搜索但未真正调工具”与“最新信息应直接搜索”的回归检查

## 适合谁更新

- 希望小虾米直接处理“今天最新 AI 资讯 / 最近动态 / 最新外部信息”这类请求的用户
- 遇到过模型说“做不到”或让用户手动执行 `/百度`、`/prismfy` 的用户
- 遇到过模型口头说会搜索、实际却卡住不动的用户
