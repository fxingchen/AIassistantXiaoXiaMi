# 小虾米 AI 助手

VS Code 侧边栏 AI 聊天插件，基于 SiliconFlow API，支持多模型切换和代码上下文感知。

## 快速开始

1. 点击左侧活动栏的 🐱 图标打开小虾米
2. 首次使用需填入 [SiliconFlow API Key](https://cloud.siliconflow.cn)（免费注册可获得额度）
3. 开始提问

## 主要功能

### 代码上下文（自动）
- 在编辑器中**选中代码** → 聊天输入框自动出现选区芯片（如 `extension.js:41-51`）
- 只是**打开文件**（无选区）→ 自动出现文件名芯片（如 `extension.js`）
- 发送消息时，芯片对应的代码会自动附加进去
- 点击芯片上的 `×` 可临时移除

### 手动附加上下文
| 操作 | 方式 |
|------|------|
| 发送选中代码 | `Ctrl+Shift+I` 或点击 📎 按钮 |
| 发送整个文件 | 点击 📄 按钮 |
| 粘贴截图 | `Ctrl+V` 直接粘贴 |
| 上传图片 | 点击 🖼 按钮或拖入输入框 |
| 拖入代码文件 | 直接拖文件到输入框 |

### 斜杠快捷指令
输入 `/` 弹出快捷菜单：

| 指令 | 说明 |
|------|------|
| `/解释` | 解释代码作用和逻辑 |
| `/修复` | 找出并修复 Bug |
| `/优化` | 优化代码性能和质量 |
| `/测试` | 生成单元测试 |
| `/审查` | 代码审查 |
| `/文档` | 生成注释和文档 |
| `/翻译` | 翻译注释为中文 |

### 其他
- **多模型切换**：顶部下拉框选择 Qwen2.5-VL-72B、DeepSeek-V3/R1 等模型
- **历史记录**：点击 📋 查看历史对话，最多保存 50 条
- **新对话**：点击 ↺ 开始新对话
- **停止生成**：生成中点击 ■ 停止

## 常见问题

**Q: 提示"API Key 无效"怎么办？**
点击 ⚙ 按钮重新输入 API Key，确认从 [cloud.siliconflow.cn](https://cloud.siliconflow.cn) 复制的是完整的 `sk-` 开头的密钥。

**Q: 选了代码但芯片没出现？**
确保代码文件已在编辑器中打开（不是预览模式），选区后稍等片刻让插件检测到。

**Q: 截图粘贴没反应？**
需要使用支持视觉能力的模型（如 Qwen2.5-VL-72B），其他纯文本模型不支持图片输入。

**Q: 历史记录在哪里存储？**
存储在浏览器 `localStorage` 中（VS Code webview 独立存储），清除插件数据会丢失历史。

**Q: API Key 安全吗？**
Key 仅存储在本地 localStorage，不会上传到任何第三方服务器。
