# 小虾米 AI 助手 v0.0.18

发布提交：678b35b

本次版本主要补强了工作区外本地目录修改的安全性和工具链路的稳定性，重点解决“修改到一半出错”和“只回复请稍等就停住”的问题。

## 亮点

- `apply_local_patch` 现在会先备份受影响文件，再执行修改
- 如果本地补丁中途失败，会自动回滚已经完成的前序修改
- 工具调用后如果模型只输出“请稍等 / 正在处理”之类的中间进度说明，前端会自动追加一次继续执行要求，减少半路停住
- 自测补充了本地补丁回滚和进度性回复续跑守卫

## 适合谁更新

- 想让小虾米直接分析并修改工作区外本地目录的用户
- 正在测试已安装软件目录修复、升级、定点改写能力的用户
- 遇到过工具调用后卡在中间状态、没有继续完成分析或修复的用户
